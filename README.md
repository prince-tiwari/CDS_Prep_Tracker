# 🦅 CDS 2026 (IMA) — 60 Day Operations Planner

> **"Discipline today, leadership tomorrow."**

A tactical, high-performance single-page web application designed for UPSC CDS (Indian Military Academy) aspirants. Originally engineered as a personal tool to crack the preparation timeline, this project is now open-sourced to help fellow defense aspirants systematically conquer their 60-day syllabus, track their daily discipline, and log live-fire mock tests.

🎯 **Target Score:** 200+ Marks

---

## ⚡ Features & Tactical Modules

*   **⚡ Mission Dashboard:** Real-time calculation of overall completion progress, daily target track (5–6 hours), and study streak counter to maintain continuous momentum.
*   **📅 Field Calendar (60-Day Syllabus):** A pre-mapped, structured 8-week curriculum covering Mathematics, English, and General Knowledge (GK) with checkable objectives.
*   **🔥 Habit Tracker:** Weekly discipline matrix for keeping tabs on daily essentials like Current Affairs, Vocabulary, Running, and Sleep.
*   **🎯 Live Fire Mock Logs:** Form tool to log your mock scores out of 300, complete with automated tracking of your historical best and running average.
*   **📝 Field Notebook:** Auto-saving quick-reference blocks for important formulas, GK facts, active vocabulary, and a mistakes log.
*   **📊 Sitrep (Statistics):** Deep-dive breakdown of subject-wise completion bars and performance history to optimize weak sectors.

---

## 🛠️ Tech Stack & Architecture

This planner is intentionally built **without external frameworks or libraries** to ensure lightning-fast execution, military efficiency, and zero operational dependency.

*   **Frontend:** Semantic HTML5 & Responsive CSS3 (Custom Grid/Table systems)
*   **Typography:** Oswald (Display), Libre Caslon Text (Body), and JetBrains Mono (Tactical Data)
*   **Logic Pipeline:** Pure Vanilla JavaScript (ES6+) enclosed inside a secure IIFE block
*   **Data Persistence:** Zero servers needed. Full structural application state is saved locally via `localStorage`.
